# Directory Overview

This directory contains the files for a simple, static website for a cleaning company called SkyClean. The website is a single landing page that showcases the company's services. It is available in both Romanian and English.

# Key Files

*   `index.html`: The main HTML file for the website. It contains the structure, content, and styling for the landing page.
*   `CNAME`: This file specifies the custom domain for the website, which is `skyclean.ro`.
*   `logo.png`: The company's logo.
*   `canapea.jpg`, `saltea.jpg`, `covor.jpg`, `mocheta.jpg`: Images used in the "services" section of the website.
*   `tricouri.png`: This image is not used on the website.

# Usage

The files in this directory are used to build the `skyclean.ro` website. The website is a simple, one-page portfolio for the cleaning company. It can be updated by editing the `index.html` file.
